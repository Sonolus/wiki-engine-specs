# `Paint`

Paint an instruction icon.

## Arguments

| Argument | Description     |
| -------- | --------------- |
| id       | Icon identifier |
| x        | X               |
| y        | Y               |
| size     | Size            |
| rotation | Rotation        |
| z        | Z               |
| a        | Alpha           |

## Return

`0`.

## Remarks

Only available in tutorial.
