# `DecrementPostShifted`

Decrement a value in block by `1`.

## Arguments

| Argument | Description      |
| -------- | ---------------- |
| id       | Block identifier |
| x        | X                |
| y        | Y                |
| s        | S                |

## Return

Value after decrement.

## Remarks

Equivalent to: `DecrementPost(id, x + y * s)`.
