# `IncrementPreShifted`

Increment a value in block by `1`.

## Arguments

| Argument | Description      |
| -------- | ---------------- |
| id       | Block identifier |
| x        | X                |
| y        | Y                |
| s        | S                |

## Return

Value before increment.

## Remarks

Equivalent to: `IncrementPre(id, x + y * s)`.
