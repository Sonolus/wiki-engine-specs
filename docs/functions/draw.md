# `Draw`

Draw a skin sprite.

## Arguments

| Argument | Description       |
| -------- | ----------------- |
| id       | Sprite identifier |
| x1       | X1                |
| y1       | Y1                |
| x2       | X2                |
| y2       | Y2                |
| x3       | X3                |
| y3       | Y3                |
| x4       | X4                |
| y4       | Y4                |
| z        | Z                 |
| a        | Alpha             |

## Return

`0`.

## Remarks

Points are in the order of bottom-left, top-left, top-right, bottom-right.
