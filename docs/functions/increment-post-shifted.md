# `IncrementPostShifted`

Increment a value in block by `1`.

## Arguments

| Argument | Description      |
| -------- | ---------------- |
| id       | Block identifier |
| x        | X                |
| y        | Y                |
| s        | S                |

## Return

Value after Increment.

## Remarks

Equivalent to: `IncrementPost(id, x + y * s)`.
