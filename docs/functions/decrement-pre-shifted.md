# `DecrementPreShifted`

Decrement a value in block by `1`.

## Arguments

| Argument | Description      |
| -------- | ---------------- |
| id       | Block identifier |
| x        | X                |
| y        | Y                |
| s        | S                |

## Return

Value before decrement.

## Remarks

Equivalent to: `DecrementPre(id, x + y * s)`.
