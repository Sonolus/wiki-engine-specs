# Engine Configuration

Engine configuration is used by Sonolus app to populate level and engine configuration interface.

## Syntax

```ts
type EngineConfiguration = {
    options: EngineConfigurationOption[]
    ui: EngineConfigurationUI
}
```

## Examples

```json
{
    "options": [
        // ...
    ],
    "ui": {
        // ...
    }
}
```
